# Inventario de Plantas Medicinales de Puerto Rico

Prototipo inicial para aprender VS Code y Git mientras se construye una base estructurada de plantas medicinales con posible implementación futura en WordPress.

## Objetivo
Separar contenido y estructura para facilitar:
- crecimiento funcional
- organización de datos
- futuras migraciones a WordPress
- uso de ramas experimentales en Git
